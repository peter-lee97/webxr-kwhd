# Interactive Voxel Cats with Feather Teaser Wand

## Goal
Implement interactive voxel cats that can independently rotate their heads and respond to a feather teaser wand held by the user. The cats should tilt their heads and perform other actions when the wand is nearby.

## Current Implementation Analysis
- Cats are implemented as voxel-based creatures in `VoxelOcelot.js`
- Each cat has a head group that can be independently animated
- Cats already have various animations: idle, walking, sitting, jumping, stretching, looking around
- Interaction system exists for cursor/controller clicks
- XR controllers and hand tracking are supported

## Plan Overview
1. Add head rotation capabilities to the VoxelOcelot class
2. Create a feather teaser wand component
3. Implement proximity detection between wand and cats
4. Add new cat behaviors for wand interaction
5. Update interaction system to handle wand proximity

## Detailed Implementation Steps

### 1. Enhance VoxelOcelot Head Rotation
- Modify `VoxelOcelot.js` to add independent head rotation capabilities
- Add methods for head tilting and rotation in X/Y/Z axes
- Add head tracking state to follow wand or other targets

### 2. Create Feather Teaser Wand Component
- Create a new component for the feather teaser wand
- Implement as a Three.js mesh with appropriate geometry (rod + feather)
- Add to XR controllers/hands when in VR mode
- Add to mouse cursor in desktop mode for testing

### 3. Implement Proximity Detection System
- Add a system to detect when wand is near cats
- Calculate distance between wand tip and cat head/body
- Define interaction radius for wand detection

### 4. Add Wand Interaction Behaviors
- Create new animation states for wand interaction:
  - Head tilting toward wand
  - Following wand movement with eyes
  - Playful pouncing when wand moves quickly
  - Curious inspection when wand is stationary
- Add new sound effects for wand interaction

### 5. Update Interaction System
- Modify existing interaction system to handle wand proximity
- Add cooldown system to prevent spamming interactions
- Update dashboard to show wand interaction status
